Manual Técnico: Creación del Sitio Web Personal

1. Objetivo

El objetivo de este proyecto fue crear un sitio web personal utilizando marcos, con un menú lateral que permite visualizar distintas secciones dentro de un área de contenido.

2. Estructura del proyecto

La estructura del sitio es la siguiente:

mi_sitio/
├── index.html              ← Página principal con menú y marco (iframe)
├── hoja_de_vida.html       ← Información personal del alumno
├── notas.html              ← Tabla con nombres y notas
├── galeria.html            ← Galería de 10 imágenes
├── video.html              ← Página con video de YouTube
├── juego.html              ← Juego en JavaScript (simple)
├── manual.html             ← Este manual técnico
└── imagenes/               ← Carpeta con las imágenes de la galería
    ├── img1.jpg
    ├── img2.jpg
    └── ...hasta img10.jpg

3. Paso a paso de creación

  1. Se creó el archivo index.html con diseño de dos columnas: un menú lateral con enlaces y un <iframe> donde se muestra el contenido de las otras páginas.
  2. Se crearon páginas individuales para cada sección del menú:
    • hoja_de_vida.html: contiene datos personales del alumno.
    • notas.html: tabla con cursos y notas del primer y segundo parcial.
    • galeria.html: contiene 10 imágenes almacenadas en una carpeta imagenes/.
    • video.html: contiene un video incrustado de YouTube.
    • juego.html: contiene un pequeño juego en JavaScript (Adivina el número).
    • manual.html: explica cómo se creó el sitio.
  3. Se usaron estilos CSS básicos para mejorar la presentación.
  4. Las imágenes de la galería fueron agregadas en una carpeta y referenciadas con <img src="imagenes/...">.
  5. El sitio puede subirse a un hosting gratuito como GitHub Pages o Netlify, incluyendo el número de carnet en la URL.

4. Lenguajes y herramientas utilizados

  • HTML5 para la estructura del sitio.
  • CSS para estilos básicos.
  • JavaScript para interacción en el juego.
  • GitHub Pages (opcional) para publicar el sitio.

5. Consideraciones finales

El uso de marcos mediante <iframe> permite mostrar múltiples páginas dentro de una sola sin recargar toda la web. Esta técnica es útil para sitios simples, pero en proyectos más grandes se recomienda usar otras estructuras como componentes o SPA.
